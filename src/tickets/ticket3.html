Билет 3

Состояния квантовой системы, чистые и смешанные состояния, статистический оператор, и вероятностное толкование измерения.

Структура билета:

  1. Понятие состояния (Квантовая система, Вектор состояния, Волновая функция, Наблюдаемая величина).
  2. Чистые и смешанные состояния.
  3. Физический смысл волновой функции (Разложение по спектру).
  4. Среднее значение и дисперсия физической величины.
  5. Статистический оператор (матрица плотности) смешанного и чистого состояний.
  6. Вероятностное толкование процесса измерения и коллапс волновой функции.

1. Понятие состояния

  • Квантовая система: физическая система, которой математически ставится в соответствие гильбертово пространство $\mathcal{H}$.
  • Состояние: математический объект, который полностью задает вероятности всех возможных результатов любых измерений над системой.
  • Волновая функция (ВФ): вектор $|\psi\rangle \in \mathcal{H}$, определяющий состояние квантовой системы. Обычно состояния отождествляются с волновой функцией. Она всегда нормирована на единицу: $\langle \psi | \psi \rangle = 1$.
  • Наблюдаемая величина: математически описывается линейным самосопряженным (эрмитовым) оператором $\hat{A}$. Область определения $D(\hat{A}) \subset \mathcal{H}$. Все возможные результаты измерения этой величины лежат в спектре оператора $\sigma(\hat{A}) \subset \mathbb{R}$.

2. Чистые и смешанные состояния

Определения чистого и смешанного состояния ▼
Чистое состояние: Состояние, которое описывается одной волновой функцией $|\psi\rangle$ (или $\psi(q)$ в координатном представлении). Это состояние максимального знания о системе.
Смешанное состояние: Состояние, в котором система с вероятностью $w_1$ находится в чистом состоянии $|\psi_1\rangle$, с вероятностью $w_2$ — в состоянии $|\psi_2\rangle$, и т.д. Описывается ансамблем $\{w_n, |\psi_n\rangle\}$, где $\sum_k w_k = 1$.

Важно: В смешанном состоянии нет квантовой интерференции между разными $|\psi_k\rangle$. Мы просто не знаем, в каком именно из них находится система (классическая неопределенность).

Примеры смешанных состояний

  • Пучок неполяризованных частиц (фотонов или электронов). Мы не можем описать весь пучок одной волновой функцией с определенным спином, так как частицы в пучке имеют разные случайно распределенные спины.
  • Квантовая подсистема, взаимодействующая с большей системой (термостатом). Даже если вся замкнутая Вселенная находится в чистом состоянии, её выделенная часть из-за квантовой запутанности будет находиться в смешанном состоянии.

3. Физический смысл волновой функции

Физический смысл ВФ заключается в её вероятностной интерпретации. Разложим волновую функцию по собственным векторам некоторого оператора наблюдаемой $\hat{L}$ (базис $|\phi_n\rangle$).

Дискретный спектр

Единичный оператор: $\hat{I} = \sum_n |\phi_n\rangle\langle\phi_n|$. Тогда разложение:

$$|\psi\rangle = \sum_n c_n |\phi_n\rangle, \quad \text{где } c_n = \langle \phi_n | \psi \rangle$$

Вероятность получить невырожденное собственное значение $\lambda_n$ при измерении равна квадрату модуля коэффициента разложения:

$$P_n = |\langle \phi_n | \psi \rangle|^2 = |c_n|^2$$

Непрерывный спектр (на примере координаты $\hat{x}$)

Для оператора координаты $\hat{x} \psi_\lambda = \lambda \psi_\lambda$, собственные функции $\psi_\lambda(x) = \delta(x-\lambda)$.

Аналогом вероятности $P_n$ выступает плотность вероятности $P(\lambda)$:

$$P(\lambda) = |\langle \psi_\lambda | \psi \rangle|^2 = \left| \int_{-\infty}^\infty dx\, \delta(x-\lambda)\psi(x) \right|^2 = |\psi(\lambda)|^2$$

Таким образом, $|\psi(x)|^2$ — это плотность вероятности обнаружить частицу в точке $x$. Квадрат модуля ВФ задает распределение вероятностей в непрерывном спектре.

4. Среднее значение и дисперсия

Математическое ожидание (Среднее значение)

Среднее значение физической величины $L$ в чистом состоянии $|\psi\rangle$ вычисляется как:

$$M[L] = \overline{L} = \sum_n P_n \lambda_n = \sum_n \langle \psi | \phi_n \rangle \lambda_n \langle \phi_n | \psi \rangle = \langle \psi | \hat{L} | \psi \rangle$$

В случае непрерывного спектра (интегральный вид в $L_2$):

$$M[L] = \int_{-\infty}^\infty \lambda \rho(\lambda) d\lambda = \int_{-\infty}^\infty \psi^*(x)\hat{L}\psi(x) dx$$

Дисперсия (Среднеквадратичное отклонение)

Дисперсия характеризует разброс (ширину распределения) результатов измерений вокруг среднего значения:

$$\sigma^2(\hat{L}) = M[(\hat{L} - \overline{L})^2] = M[\hat{L}^2] - (M[\hat{L}])^2 \ge 0$$

Среднеквадратичное отклонение (неопределенность величины $L$):

$$\Delta L = \sqrt{\sigma^2(\hat{L})} = \sqrt{\overline{L^2} - \overline{L}^2}$$

5. Статистический оператор (Матрица плотности)

В квантовой статистической механике вместо волновых функций удобнее использовать универсальный объект — статистический оператор $\hat{\rho}$.

Для смешанного состояния ▼

Если система находится в ансамбле состояний $\{w_k, |\phi_k\rangle\}$, то статистический оператор по определению:

$$\hat{\rho} = \sum_k w_k |\phi_k\rangle\langle\phi_k|$$

Среднее значение физической величины $L$ в смешанном состоянии ищется через след (Trace):

$$M[L] = \text{Tr}(\hat{\rho}\hat{L})$$
Для чистого состояния ▼

Чистое состояние является частным случаем смешанного, когда одна из вероятностей $w_1 = 1$, а все остальные равны нулю. Тогда:

$$\hat{\rho} = |\psi\rangle\langle\psi|$$

Этот оператор является проектором на состояние $|\psi\rangle$. Для чистого состояния всегда выполняется свойство идемпотентности: $\hat{\rho}^2 = \hat{\rho}$. Для смешанного это не так ($\text{Tr}(\hat{\rho}^2) < 1$).

6. Вероятностное толкование измерения и коллапс ВФ

Процесс измерения и Постулат Фон Неймана

Измерение в квантовой механике активно воздействует на систему. Пусть мы измеряем наблюдаемую $\hat{L}$. Согласно проекционному постулату фон Неймана:

  1. При измерении мы всегда получаем одно из собственных значений $\lambda_n$ оператора $\hat{L}$. Вероятность этого исхода: $P_n = |\langle\phi_n|\psi\rangle|^2$.
  2. Коллапс (редукция) волновой функции: Сразу после того, как прибор показал значение $\lambda_n$, исходное состояние $|\psi\rangle$ мгновенно "схлопывается" (переходит) в соответствующее собственное состояние $|\phi_n\rangle$.
  3. Если мы повторим измерение $\hat{L}$ через бесконечно малый промежуток времени, мы со $100\%$ вероятностью снова получим $\lambda_n$.
Измерение вырожденного состояния:
Если полученное собственное значение $\lambda_n$ вырождено (ему соответствует не одна функция, а целое подпространство с базисом $|\phi_{n,k}\rangle$), то волновая функция проецируется на всё это подпространство.
Новое состояние: $$|\psi'\rangle = \frac{\hat{P}_\lambda |\psi\rangle}{\|\hat{P}_\lambda \psi\|}$$ Где $\hat{P}_\lambda = \sum_k |\phi_{n,k}\rangle\langle\phi_{n,k}|$ — оператор проектирования на собственное подпространство $\lambda_n$.
